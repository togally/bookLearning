## 资源下载

```Plain Text
# Download release from the Apache mirror
$ wget https://dist.apache.org/repos/dist/release/rocketmq/5.1.4/rocketmq-all-5.1.4-bin-release.zip

# Unpack the release
$ unzip rocketmq-all-5.1.4-bin-release.zip
```

## Local模式部署

```Plain Text
# 启动nameSrv
nohup ./mqnamesrv > ../log/namesrv.log &

# 启动broker
nohup sh ./mqbroker -n localhost:9876 > ../log/mqbroker.log &

# 启动proxy 不知道为何启动会报错
nohup ./mqproxy -n 127.0.0.1:9876 > ../log/mqproxy.log &
```
